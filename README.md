# SALES-DASHBOARD
Project Overview
This project presents an interactive Sales Data Analysis Dashboard built using Microsoft Power BI.  
The objective of this project is to analyze sales performance, identify key business trends, and provide data-driven insights for business decision-making.

Business Objectives
- Analyze monthly and yearly sales trends
- Identify top-performing products and regions
- Evaluate profit margins
- Compare category-wise performance
- Track KPIs for business growth

Dataset Information
Structured sales dataset including:
- Order Date
- Region
- Product Category
- Sales
- Profit
- Quantity

Key Insights Generated
- Identified peak sales months and seasonal trends
- Determined highest revenue-generating product categories
- Analyzed profit contribution by region
- Highlighted underperforming segments for improvement

Tools and Technologies Used
- Microsoft Power BI
- Data Cleaning and Transformation
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

Dashboard Features
- Interactive Filters and Slicers
- KPI Cards
- Sales Trend Line Charts
- Category-wise Bar Charts
- Region-wise Performance Analysis
- Drill-down functionality
